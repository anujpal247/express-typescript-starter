# Template setup

1. clone project

```
  git clone https://github.com/anujpal247/express-typescript-starter.git <project-name>
```

2. change dir

```
  cd <project-name>
```

3. install dependencies

```
  npm i
```

4. create `.env` file inside root dir and add

```
  PORT=8080
```

5. start express server

```
  npm run dev
```
